Chaining modifiers

The output of one modifer can be used as the input for the next one. Seen the movie The Human Centipede? It's a bit like that.

Let's rummage around in the bag of contrived examples and see what comes out... yep... this one will do... it's a shopping basket.

To make the template work, we first of all need to add a currency formatter. Here's one for the UK (if you're not from here, write one for your own currency for extra credit):

function ( num ) {
  if ( num < 1 ) return ( 100 * num ) + 'p';
  return 'Â£' + num.toFixed( 2 );
}

Add that to modifiers as formatCurrency, then update the first <td> to use the new modifier:

<td>{{price | formatCurrency}}</td>

Execute the code. The price should change from '0.49' to '49p'.

To show the total, we could calculate a separate total value. But for this example we'll use a modifier instead. Add a modifier called timesQty:

function ( num ) {
  return this.get( 'qty' ) * num;
}

Then chain the timesQty and formatCurrency modifiers:

<td>{{price | timesQty | formatCurrency}}</td>

Execute the code.
