Making it sortable (part 1)

It's time to make our table sortable. We've added a 'sortable' class to the three headers to indicate they can be clicked on.

First, let's add a proxy sort event to each column header:

<th class='sortable' proxy-tap='sort'>Superhero name</th>
<th class='sortable' proxy-tap='sort'>Real name</th>
<th class='sortable' proxy-tap='sort'>Superpower</th>

That way, when the user taps one of the column headers, the view will fire a sort event.

Next, we add a data-column property indicating which column each header belongs to.:

<th class='sortable' proxy-tap='sort' data-column='name'>Superhero name</th>
<!-- and so on... -->

That way, we can retrieve the column ID within our handler, and sort the list accordingly:

ractive.on( 'sort', function ( el, event ) {
  var sortColumn = el.getAttribute( 'data-column' );
  alert( 'Sorting by ' + sortColumn );
});

Execute the code. When you tap on the three sortable headers, the browser should alert the name of the column we're sorting by. Now we just need to add the sorting logic.
