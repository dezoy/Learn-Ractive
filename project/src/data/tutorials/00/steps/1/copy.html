You're right, it's not. Let's make our #template more templatey – replace the hard-coded text with some variables:

<p>{{greeting}} {{recipient}}!</p>

Then, add some data to it, by adding a data initialisation option to our #javascript so that the code looks like this:

var ractive = new Ractive({
  el: output,
  template: template,
  data: { greeting: 'Hello', recipient: 'world' }
});

Execute the code (click in the #javascript box, hit Shift-Enter). It should look exactly as it did before.
